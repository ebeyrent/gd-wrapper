      Creates a new blank image resource.

      __construct(int $width, int $height) 

      IMPORTANT: to guarantee that two Resource objects do not have the same image resource, this constructor creates a copy of $resource parameter and only then assigns the copy to $raw attribute.

      Functions that creates resources JUST to be used in theese objects SHOULD destroy it right after instanciate an object to prevent memory leaks.
